## Overview

FPGA-accelerated neural inverse kinematics engine for real-time 6-DOF robotic manipulation using Verilog, Xilinx Artix-7, and embedded motor control.

## Related Work

Li, Y., Li, L., and Choset, H.
"FPGA Acceleration for High Dimensional Inverse Kinematics"
(ShanghaiTech University / Carnegie Mellon University)

This project extends the concept of FPGA-based inverse kinematics acceleration by investigating neural-network inference acceleration for real-time robotic manipulation and comparing performance against conventional Jacobian-based approaches.
